Smart Expense Tracker

A "simple and user-friendly web app" to track your daily expenses. Add, delete, clear, and download your expenses in a CSV file. Works on both desktop and mobile.


ðŸ”¹ Features

- Add expenses with Amount, Category, and Note
- View all added expenses with date and time
- Delete individual expenses
- Clear all expenses with a single click  
- Total amount spent automatically calculated  
- Download expenses as CSV file for records  
- Fully user-friendly and works offline (stores data in browser)


ðŸ”¹ How to Use

1. Open the app in your browser: [GitHub Pages Link](https://joyrebeccamadari.github.io/Smart-Expense-Tracker/)  
2. Enter "Amount", "Category", and an optional "Note"
3. Click "Add Expense" to save  
4. See all expenses listed below with date and time  
5. Click "Delete" to remove a single expense  
6. Click "Clear All" to delete all expenses  
7. Click "Download CSV" to save your expenses

- HTML for structure  
- CSS for styling  
- JavaScript for functionality and data storage (localStorage)  


ðŸ”¹ Notes
